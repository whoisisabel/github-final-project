# Contributing to github-final-project

Thank you for your interest in contributing! All contributions, bug reports, bug fixes, documentation improvements, enhancements, and ideas are welcome.  

We follow a collaborative and respectful approach to make this project better for everyone.

## How to Contribute

### 1. Reporting Bugs
- Search the [issues](https://github.com/whoisisabel/github-final-project/issues) to see if the bug has already been reported.
- Open a new issue if it hasn't been reported yet.
- Provide a clear description of the problem, steps to reproduce it, and screenshots if possible.

### 2. Suggesting Enhancements
- Open an issue with a clear title and description of your idea.
- Explain why it would be useful and, if possible, provide examples or references.

### 3. Submitting Code
1. Fork the repository.
2. Create a new branch for your work:
   ```bash
   git checkout -b feature/your-feature-name
